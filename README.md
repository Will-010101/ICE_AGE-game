# Ice Age Laboratory Game

## Overview

Ice Age is a C# educational game developed for the National Research Centers Software Competition.

The project was designed to provide an engaging learning experience by presenting laboratory concepts through interactive gameplay and educational challenges.

## Achievement

🥈 **2nd Place – District Level**  
**National Research Centers Software Competition**

The project received second place at the district level for its educational design and implementation.

## Technologies

- C#
- .NET Framework
- Windows Forms

## Features

- Interactive educational gameplay
- Laboratory concept learning
- User-friendly Windows desktop interface
- Game-based learning experience
